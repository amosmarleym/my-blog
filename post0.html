Basics of Webapplication Database Intrusion and Vulnerability Pentesting

by Amos Adagboyi

Posted on October 15, 2022 at 10:00 PM

Web application Database Intrusion and penetration testing is a very important aspect of cybersecurity every web application developer should be conversant with especially backend developers and every beginner cybersecurity analyst. There are specific tools which are used for database intrusion and vulnerability testing, these tools are usually paired with google dorks or google dork searching for vulnerable websites. Databases hold a variety of sensitive information like Credit card numbers, Travel passports and Idenity cards, Social security numbers, Email ID and password. these personal information can be used to cause irreperaple damage and loss of reputaion if a websites database is breached by attackers and information gets into the wrong hands. I will share some of the most effective tools used in databse hacking vulnerability testing and ways of preventing a database breach.

.

User Interface SQLi Dumper

(1)Sqli dumper is one of the most effective tools used by penetration testers and malicious attackers for sql injection and database Intrusion. Sqli dumper is easy to use, once you put in a list of google dorks and vulnerable sites, it scans them and starts dumping the content of the database













User Interface Sqlmap

(2)Sqlmap is one of the best tool for atutomated Sql injection vulnerabvility scanning. this tool is opensource and written in python, you will need to install verison 2.7 of python or higher depending on the version of sqlmap you have. Sqlmap has many features and supports many databases,Sqlmap is a command line tool.
















Image showing a list of google dorks

Google dorks are strings used for looking vulnerable websites on the web. Google Dorking, is a computer hacking technique that uses Google Search and other Google applications to find security holes in the configuration and computer code that websites use.
